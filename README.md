# Eid Mubarak Card - Contest Version

A vibrant, interactive Eid greeting card built with HTML5, CSS3, and Vanilla JavaScript. Features include animated mosques, falling petals, sound effects, and a mobile-banking Salami sender.

## 🚀 Features
- **Interactive Pages**: Navigate between greetings, solidarity messages, and checklists.
- **Selectable Text**: Optimized for sharing; highlight and copy any message or poem.
- **Save & Share**: Download cards as images or share via WhatsApp/Facebook.
- **Salami Sender**: Deep links to bKash and Nagad for sending Eid gifts.
- **Atmospheric UI**: Animated moon, lanterns, and chimes.

## 🛠️ Tech Stack
- **Structure**: HTML5 Semantic Elements
- **Styling**: Vanilla CSS (Custom Variables, Flexbox, Animations)
- **Logic**: Vanilla JavaScript (Web Audio API, localStorage, Canvas)
- **Library**: `html2canvas` (for card downloads)

## 📁 Project Structure
- `index.html`: The main application file.
- `eid-contest-final.html`: Submission-ready contest version.

## 📖 Local Development
To view the project locally, you can use any static file server:
```bash
# Example using npx
npx serve . -p 3000
```
Open `http://localhost:3000` in your browser.
